# Freeplay Album Data Schema Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3]
### Added
- Added `albumOSTName` to change the OST at the top right of the Freeplay menu.

## [1.0.2]
### Added
- Added `albumTitleOffsets` to change the horizontal and vertical position of the album title.

## [1.0.1]
### Added
- Added `albumTitleAnimations` to modify animation data for the album art.

## [1.0.0]
Initial release.
